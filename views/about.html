This app was developed by Saurabh Kumar to visualize and analyze trends of graduate admits and rejects to various universities.

This involved:

  1. Scraping:
    Scrape the data from the grad cafe
  2. Parsing:
    Parse the raw html and separate into individual components
  3. Entity Resolution:
    Resolve synonymous universities (Carnegie Mellon, Carnegie Mellon University, CMU, etc. all correspond to the same entity) and create a final set of university application entries.
  4. Storing:
    Store these in Mongodb making it easier to query on various keys
  5. Query and Render:
    Execute queries and render on the client
